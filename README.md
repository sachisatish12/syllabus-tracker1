# 🎓 Syllabus Tracker

A modern web application built with Next.js that helps students organize and track their course syllabi, assignment weights, due dates, and office hours in one centralized dashboard.

---

## ✨ Features

- 📚 Add and manage classes
- 👩‍🏫 Track professor information
- 🧾 Add assignments with weight percentages
- 📅 Optional due dates for assignments
- 🕒 Separate office hours tracking per class
- 📊 Clean dashboard UI with structured cards
- 🎨 UGA-inspired theme (red, black, white)

---

## 🖥️ Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Git & GitHub
